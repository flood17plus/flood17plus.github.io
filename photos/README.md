# Фотографии для медальонов

Положите сюда 3 изображения и переименуйте в `1.jpg`, `2.jpg`, `3.jpg` — они автоматически подставятся в медальоны.

## Порядок
- `1.jpg` — левый медальон
- `2.jpg` — центральный (крупный)
- `3.jpg` — правый

## Рекомендации
- Квадратные изображения (фото будет обрезано по кругу)
- Минимум 400×400 px, лучше 800×800
- Форматы: `.jpg`, `.png`, `.webp`

Если файла нет — вместо фото автоматически показывается рунический placeholder.
Хотите другой формат (png, webp) — откройте `index.html` и поправьте `src="photos/1.jpg"` на нужное расширение.
